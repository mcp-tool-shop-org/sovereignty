# Cozy Postcard Example

A 3-player Campfire game with the cozy recipe. Everyone chose
different goals and the game ended close.

```
╭─────────────── Campfire Postcard ───────────────╮
│  Sovereignty: Campfire                          │
│  Round 8 | Ada, Ben, Cleo                       │
│                                                 │
│  Ada: 12c 5r 2u | Ben: 8c 7r 1u | Cleo: 6c 9r 0u │
│                                                 │
│  Proof: sha256:a3f8c1...                        │
│                                                 │
│  Kind: Ben helps Cleo at Help Desk (+1 Rep each)│
│  Brave: Ada apologizes to Ben. -1c, +1 Rep.     │
│  Toast: Cleo toasts Ben: "good neighbor."       │
╰───────── sov postcard --style cozy ─────────────╯
```

**What happened:** Ada was racing for Prosperity (20 coins) and
stepped on some toes early. Ben kept helping people at the Help Desk
and quietly built reputation. Cleo picked Beloved (10 rep) and got
there first — but barely. The Toast from Cleo to Ben was genuine.

**Why this postcard works:** Three different strategies, one
apology that mattered, and a toast that landed. The cozy style
filters out the noise and shows the moments of connection.
