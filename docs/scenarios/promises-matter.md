# Promises Matter

> Tonight, your word is everything. Every deal is a handshake. Every
> voucher is a debt. Every promise you make out loud will come back
> around — as reputation earned or reputation lost. The table remembers.

| Setting | Value |
|---------|-------|
| Tier | Campfire |
| Recipe | promise |
| Players | 2-3 |
| Rounds | 8-10 |
| Time | 30 min |

## What to expect

The promise recipe loads the deck with commitment-heavy cards: Awkward
Favor, Supply Run, Spot Me, Builder's Promise. The voucher pile is
thick — Small Loans, Big Loans, Blank Vouchers, Emergency Loans. Every
few turns, someone will owe someone something.

Events lean toward trust and consequences: Rumor (will someone vouch
for you?), Swindle (forced voucher collection), Trust Crisis (low Rep
gets worse). The table will spend more time negotiating terms than
counting coins.

This is a great scenario for 2-3 players who already know the basics.
Smaller tables mean every promise is personal. You can't hide in a crowd.
Recommend the **Beloved** win condition (10 Rep) for at least one player.

## Start command

```bash
sov new --recipe promise -p Alice -p Bob -s 1
```

## After the game

Run `sov postcard --style spicy` to highlight broken promises and trades.

Debrief question: *"Was there a promise you almost broke? What stopped you?"*
