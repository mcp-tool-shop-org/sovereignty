# Economic Postcard Example

A 4-player Town Hall game where the market told the real story.
Wood ran out and the table scrambled.

```
╭──────────────── Town Hall Postcard ────────────────╮
│  Sovereignty: Town Hall                            │
│  Round 12 | Ivy, Jay, Kit, Lux                     │
│                                                    │
│  Ivy: 11c 4r 2u 2F | Jay: 8c 5r 3u 1W             │
│  Kit: 14c 3r 1u | Lux: 6c 7r 2u 1F 1T             │
│                                                    │
│  Proof: sha256:c4a1f3...                           │
│  Anchored: https://testnet.xrpl.org/transactions/...│
│                                                    │
│  Market: Wood scarce (supply 1, price 5)           │
│  Buy: Jay buys Wood for 4 coins                    │
│  Sell: Kit sells Food for 3 coins                  │
╰────────── sov postcard --style economic ───────────╯
```

**What happened:** By round 6, a Logging Ban event pushed Wood
supply to 2. Jay saw it coming and bought early. By round 9, Wood
hit supply 1 and price 5 — the most expensive resource the table
had ever seen. Kit hoarded coins and skipped resources entirely,
going for pure Prosperity. Lux built steadily with mixed resources
and ended up closest to Builder (4 upgrades).

**Why this postcard works:** The economic style shows the market
as a character. The Wood scarcity is the story — it changed how
everyone played. The anchored proof means this game's history is
on the XRPL Testnet permanently.
