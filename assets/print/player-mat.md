|            [ ] Beloved (10 rep)                  |
|            [ ] Builder (4 upgrades)              |

|                                                  |
|  PROMISES THIS ROUND                             |
|  "I promise ________________________________"    |
|  Kept? [ ] +1 Rep   Broke? [ ] -2 Rep            |

|  Rep >= 8: Trade with 2 players per turn          |
|                                                  |
