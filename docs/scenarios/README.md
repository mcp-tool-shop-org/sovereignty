# Scenario Packs

**Zero new rules. Just vibes.**

A scenario pack is a curated combination of tier, recipe, and table norms
that shapes the feel of your session. Pick one, read the intro aloud, and play.

## Available scenarios

| Scenario | Tier | Recipe | Players | Time |
|----------|------|--------|---------|------|
| [Cozy Night](cozy-night.md) | Campfire / Market Day | cozy | 2-4 | 30-45 min |
| [Market Panic](market-panic.md) | Town Hall | market | 3-4 | 45-60 min |
| [Promises Matter](promises-matter.md) | Campfire | promise | 2-3 | 30 min |
| [Treaty Night](treaty-night.md) | Treaty Table | — | 3-4 | 75-90 min |

## How to use

1. Pick a scenario from the table above.
2. Read the vibe intro aloud before dealing cards.
3. Run the start command shown in the scenario doc.
4. Play normally. The recipe filters the deck to match the mood.

That's it. The scenario is guidance, not enforcement. The table is still authoritative.

## Make your own

A scenario is just:
- A tier
- A recipe tag (optional)
- A player count and session length
- A paragraph that sets the tone

Write it down, share it, play it. No code needed.

## From the console

```bash
sov scenario list    # see all scenarios at a glance
```
