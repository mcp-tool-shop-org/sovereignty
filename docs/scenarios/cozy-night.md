# Cozy Night

> A quiet evening with good people. Warm drinks, easy laughs, and a few
> coins changing hands. Nobody gets hurt tonight. Help each other, toast
> the good moves, apologize for the bad ones, and let the game be a
> conversation — not a competition.

| Setting | Value |
|---------|-------|
| Tier | Campfire or Market Day |
| Recipe | cozy |
| Players | 2-4 |
| Rounds | 8-10 |
| Time | 30-45 min |

## What to expect

The cozy recipe pulls the sharp edges off the deck. You'll draw events
like Boom Town, Community Dinner, and Harvest Moon — moments that
reward generosity and collective action. The spicy cards (Trust Crisis,
Swindle, Drought) sit this one out.

This is the best scenario for a first game or a mixed group. New players
get permission to be kind without feeling like they're losing. Experienced
players get to practice the Offer and the Toast — two mechanics that
reward social play.

Watch for the first Toast. It usually changes the energy at the table.

## Start command

```bash
# Campfire (simpler, no resources)
sov new --recipe cozy -p Alice -p Bob -p Carol -s 1

# Market Day (adds resources, still gentle)
sov new --tier market-day --recipe cozy -p Alice -p Bob -p Carol -s 2
```

## After the game

Run `sov postcard --style cozy` to highlight help and apologies.

Debrief question: *"What's one thing someone did tonight that you'd
want in a real neighbor?"*
